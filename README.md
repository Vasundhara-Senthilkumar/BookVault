# 📚 BookVault – CRUD Web Application

## 🚀 Project Overview

BookVault is a simple and interactive **CRUD (Create, Read, Update, Delete)** web application built using **HTML, CSS, and JavaScript**.
It allows users to manage a collection of books by adding, viewing, editing, and deleting records.

This project demonstrates core frontend development concepts such as **DOM manipulation**, **event handling**, and **client-side data storage**.

---

## ✨ Features

* ➕ Add new books with details
* 📖 View list of all books
* ✏️ Update existing book information
* ❌ Delete books from the list
* 💾 Data stored using browser storage (localStorage) *(if implemented)*

---

## 🛠️ Tech Stack

* **HTML** – Structure of the application
* **CSS** – Styling and layout
* **JavaScript** – Functionality and logic

## ⚙️ How It Works

1. User enters book details in the form
2. JavaScript captures the input and stores it
3. Data is dynamically displayed using DOM manipulation
4. Users can update or delete books anytime
